[IJulia](http://nbviewer.ipython.org/github/ipython/front-to-back/blob/master/IJulia%20Preview.ipynb),
the
[IHaskell](http://nbviewer.ipython.org/github/ipython/front-to-back/blob/master/IHaskell.ipynb),
